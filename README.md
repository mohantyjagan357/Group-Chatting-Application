## рҹ’¬ What Is This?

**Group-Chatting-Application** is a real-time chat system built entirely in **Java** using **TCP Sockets** and **Multi-threading**. It follows a classic **Server-Client architecture** вҖ” a central server manages all connections, and every message sent by any client is instantly broadcast to all other connected participants in the group.

This project demonstrates core Java networking concepts вҖ” `ServerSocket`, `Socket`, `InputStream/OutputStream`, `Thread` вҖ” along with **MySQL** for optional message persistence and **Spring Boot** for the backend layer.

## рҹҡҖ Getting Started

### Step 1 вҖ” Clone the Repository
```bash
git clone https://github.com/mohantyjagan357/Group-Chatting-Application.git
cd Group-Chatting-Application
```

### Step 2 вҖ” Set Up MySQL Database
```sql
CREATE DATABASE chat_db;
```

### Step 3 вҖ” Configure `application.properties`
```properties
spring.datasource.url=jdbc:mysql://localhost:3306/chat_db
spring.datasource.username=YOUR_MYSQL_USERNAME
spring.datasource.password=YOUR_MYSQL_PASSWORD
spring.jpa.hibernate.ddl-auto=update
spring.jpa.show-sql=true
server.port=8080
chat.server.port=9090
```

### Step 4 вҖ” Build the Project
```bash
./mvnw clean package -DskipTests    # Linux / Mac
mvnw.cmd clean package -DskipTests  # Windows
```

### Step 5 вҖ” Start the SERVER First рҹ”‘

> **Critical: Always start the server before any clients!**

```bash
# Run the server
java -cp target/group-chatting-*.jar com.chat.application.server.ChatServer
```

Or run `ChatServer.java` directly from IntelliJ IDEA.

The server will start and listen for incoming client connections.

```
[SERVER] Chat Server started on port 9090
[SERVER] Waiting for clients...
```

### Step 6 вҖ” Start CLIENT(s) in Separate Terminals

Open a **new terminal for each client** and run:

```bash
java -cp target/group-chatting-*.jar com.chat.application.client.ChatClient
```

Or run multiple instances of `ChatClient.java` from IntelliJ IDEA.

```
Enter your username: Soumya
[CONNECTED] You joined the chat!
[SERVER] Soumya has joined the group рҹҺү
```

### Step 7 вҖ” Start Chatting! рҹ’¬

Type messages in any client terminal вҖ” they'll appear instantly on all other connected clients.

```
Soumya: Hey everyone! рҹ‘Ӣ
[SERVER] Ravi has joined the group
Ravi: Hi Soumya! What's up?
Soumya: Testing our Group Chat App рҹҡҖ
Ravi: This is awesome!
```

## рҹ§ө Multi-threading Design

```java
// Server accepts new connection вҶ’ spawns a thread
while (true) {
    Socket clientSocket = serverSocket.accept();
    ClientHandler handler = new ClientHandler(clientSocket, clients);
    new Thread(handler).start();  // each client = own thread
    clients.add(handler);
}

// ClientHandler reads message вҶ’ broadcasts to all
public void run() {
    String message;
    while ((message = reader.readLine()) != null) {
        broadcast("[" + username + "]: " + message);
    }
}
```
